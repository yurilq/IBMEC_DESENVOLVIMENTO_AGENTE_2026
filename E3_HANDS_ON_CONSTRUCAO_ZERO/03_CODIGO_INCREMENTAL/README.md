# 📂 03_CODIGO_INCREMENTAL

Esta pasta contém 20 passos progressivos mostrando a evolução do código do básico ao completo.

## 🎯 Objetivo

Mostrar **evolução gradual** do código, passo a passo, cada um adicionando UM conceito novo.

## 📋 Estrutura dos 20 Passos

### Fase 1: Fundamentos (Passos 1-5)
- `passo_01_hello_world.py` - Primeiro contato com LLM
- `passo_02_parametros.py` - LLM com parâmetros (temperature)
- `passo_03_funcao_simples.py` - Função Python básica
- `passo_04_pandas_basico.py` - Carregar e filtrar CSV
- `passo_05_funcao_completa.py` - Função que consulta dados

### Fase 2: Tools Manuais (Passos 6-10)
- `passo_06_tool_manual.py` - Criar Tool() manualmente
- `passo_07_agente_basico.py` - Primeiro agente com 1 tool
- `passo_08_verbose.py` - Entender ReAct (THOUGHT → ACTION)
- `passo_09_duas_tools.py` - Agente com 2 tools
- `passo_10_debug.py` - Debugar problemas comuns

### Fase 3: Decorators (Passos 11-15)
- `passo_11_decorator_simples.py` - O que é decorator
- `passo_12_decorator_log.py` - Decorator com funcionalidade
- `passo_13_tool_decorator.py` - Usar @tool
- `passo_14_refatorar_tools.py` - Simplificar código com @tool
- `passo_15_cache_simples.py` - Introduzir @lru_cache

### Fase 4: Otimização (Passos 16-18)
- `passo_16_quatro_tools.py` - 4 tools funcionando
- `passo_17_cache_completo.py` - Cache em todas tools
- `passo_18_performance.py` - Medir diferença de performance

### Fase 5: Finalização (Passos 19-20)
- `passo_19_fewshot_cot.py` - Adicionar Few-Shot + CoT
- `passo_20_agente_completo.py` - Agente v2.0 final com security

## 🚀 Como Usar

### Durante a Aula
Siga os guias das partes (01_GUIAS_ALUNO/). 
Código incremental é para **referência**, não para copiar.

### Após a Aula (Revisão)
Leia os 20 passos em ordem para entender evolução completa.

### Se Travou
Use TEMPLATES_PRONTOS/ em vez destes arquivos (mais completos).

## 📚 Diferença: Incremental vs Templates

**03_CODIGO_INCREMENTAL/ (20 passos):**
- Didático
- Mostra EVOLUÇÃO
- Cada passo adiciona 1 conceito
- Para APRENDER progressão

**02_TEMPLATES_PRONTOS/ (5 templates):**
- Prático
- Código FUNCIONAL completo
- Alinhado com cada parte da aula
- Para DESEMPACAR se travar
